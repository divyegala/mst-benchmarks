CUDA Implementations of Minimum Spanning Tree
given in Technical Report

"Large Graph Algorithms for Massively Multithreaded Architectures"
Pawan Harish, Vibhav Vineet and P.J.Narayanan.
Technical Report IIIT/TR/2009/74, 
International Institute of Information Technology-Hyderabad


Please note the given implementation requires CUDA 1.1 hardware. 
The code are not optimized, please use CUDA profiler for better optimization.
